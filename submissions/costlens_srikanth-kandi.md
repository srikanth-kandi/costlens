## Project Description

CostLens AI is an HR Cost Intelligence platform that helps organizations track and optimize meeting spend. It calculates meeting cost based on participants and duration, attributes meetings to projects using AI-powered suggestions with fallback logic, and provides dashboards for cost trends, anomalies, and project-level visibility.

---

## Key Features

1. Dashboard analytics for project and department-wise meeting costs.
2. Employee, project, and meeting CRUD with validation.
3. AI-assisted meeting-to-project attribution using Gemini with resilient fallback.
4. Anomaly detection for unusual cost and meeting patterns.
5. Production deployment with Vercel (frontend/backend) and Render PostgreSQL.
